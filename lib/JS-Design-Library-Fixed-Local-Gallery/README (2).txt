ערכת גלריית עיצוב מקומית לפרויקט JavaScript
===========================================

פותחים את index.html בדפדפן.

מה יש כאן:
- דוגמאות Bootstrap
- דוגמאות Bulma
- דוגמאות Pico CSS
- דוגמאות Bootstrap Icons
- פרויקטים התחלתיים קטנים

מה השתנה בגרסה הזאת:
- הוסר אזור המורה.
- תוקן Pico Dialog: החלון לא נפתח לבד ולא חוסם את הדף.
- כל הדפים מיועדים ישירות לתלמידות.

קבצי ספריות שצריך להכניס ידנית:
Bootstrap:
libs/bootstrap/css/bootstrap.rtl.min.css
libs/bootstrap/js/bootstrap.bundle.min.js

Bulma:
libs/bulma/bulma.min.css

Pico:
libs/pico/pico.min.css

Bootstrap Icons:
libs/bootstrap-icons/bootstrap-icons.css
libs/bootstrap-icons/fonts/bootstrap-icons.woff
libs/bootstrap-icons/fonts/bootstrap-icons.woff2
